# Configuration de l'équipe

## Vault Obsidian
```
OBSIDIAN_VAULT_PATH=
```
Laisser vide si pas de vault sur ce projet.
Exemple : `OBSIDIAN_VAULT_PATH=D:\Perso\AppMobileMusic\docs`

## Git
```
GIT_ENABLED=true
```

## Langue
```
LANG=fr
```

## Tests
```
TEST_FRAMEWORK=
```
Framework de test utilisé par le projet (ex: `jest`, `vitest`, `pytest`, `flutter_test`).
Laisser vide si aucun framework de test n'est en place — le Testeur sera zappé automatiquement.

## Limites
```
MAX_ITERATIONS=3
```
Nombre maximum d'itérations de correction par agent avant escalade au dirigeant.
